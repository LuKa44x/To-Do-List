# 📝 To-Do List App

A modular and dynamic To-Do List application built with vanilla JavaScript, using Webpack for project bundling and organization. Users can create projects and add/manage tasks inside them — with support for due dates, priorities, and completion status.

---

## 🚀 Features

- ✅ Create and delete projects
- ✅ Add, edit, and remove tasks inside each project
- ✅ Tasks have title, description, due date, priority, and status
- ✅ Dynamic DOM rendering with JavaScript
- ✅ Fully modular code using ES6 modules
- ✅ Webpack bundled

---

## 🛠️ Technologies

- JavaScript (ES6+)
- HTML5 + CSS3
- Webpack
